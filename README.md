# Sistema de Producción en C

Proyecto desarrollado en lenguaje C utilizando programación modular.

## Características

- Uso de arreglos y punteros
- Programación modular
- Validaciones de datos
- Menú interactivo
- Cálculo de producción
- Edición y eliminación de productos

## Archivos del proyecto

- main.c
- SistemaProduccion.h
- SistemaProduccionFunciones.c
- Validaciones.h

## Validaciones implementadas

- Máximo 5 productos
- Validación de nombres largos
- Validación de cantidades positivas
- Control de inventario vacío
- Prevención de productos repetidos
- Confirmación al eliminar

## Cómo ejecutar

1. Compilar:

```bash
gcc main.c SistemaProduccionFunciones.c -o sistema
```

2. Ejecutar:

```bash
./sistema
```
